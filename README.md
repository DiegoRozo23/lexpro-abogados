# LexPro Abogados

Sistema de gestión para despacho de abogados.

## Despliegue

La aplicación se despliega automáticamente en GitHub Pages:
[https://DiegoRozo23.github.io/lexpro-abogados/](https://DiegoRozo23.github.io/lexpro-abogados/)

## Desarrollo Local

1.  Instalar dependencias:
    ```bash
    npm install
    ```
2.  Iniciar servidor de desarrollo:
    ```bash
    npm run dev
    ```

## Tecnologías

-   Next.js 15
-   Tailwind CSS
-   Shadcn UI
-   Lucide React
